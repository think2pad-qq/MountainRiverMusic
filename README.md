# 山川有乐

一个用于发布原创音乐作品和音乐相关文章的个人网站。

## 🎵 项目简介

"山川有乐"是一个专注于音乐创作与分享的个人网站，主要功能包括：

- **音乐发布**：展示和播放原创音乐作品
- **文章分享**：发布与音乐创作、音乐理论、音乐故事相关的文章
- **作品展示**：分类展示不同风格和主题的音乐作品
- **个人品牌**：建立音乐创作者的个人形象和作品集

## ✨ 功能特点

- 响应式设计，适配各种设备屏幕
- 音乐播放器集成
- 文章分类与标签系统
- 作品时间线展示
- 联系方式页面

## 🚀 快速开始

### 安装依赖

```bash
# 克隆项目
git clone https://github.com/your-username/MountainRiverMusic.git
cd MountainRiverMusic

# 安装依赖（根据实际技术栈调整）
npm install
```

### 运行开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```
